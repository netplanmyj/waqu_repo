# スクリーンショット撮影ガイド

**撮影日**: 2025年10月18日

## 📱 必要なスクリーンショット

Play Storeに必要なスクリーンショット（最低2枚、推奨4-8枚）

### 推奨する画面構成

1. **ホーム画面（メール送信画面）** 🌟 必須
   - 測定時刻と残留塩素の入力フィールド
   - 送信ボタン
   - 送信ステータスメッセージ

2. **設定画面** 🌟 必須
   - メールアドレス設定
   - デバッグモード切替
   - その他の設定項目

3. **送信履歴画面**
   - 過去の送信履歴リスト
   - 送信日時、状態表示

4. **サインイン画面**（オプション）
   - Googleサインイン画面
   - アプリロゴ表示

---

## 🎯 撮影方法

### 方法1: 実機で撮影（推奨）

#### Android端末での撮影
1. アプリを起動
2. 各画面に移動
3. **電源ボタン + 音量下げボタン** を同時押し
4. スクリーンショットが保存される
5. PCに転送

#### 撮影時の注意点
```
✅ 通知をOFF（Do Not Disturb）
✅ Wi-Fi接続中
✅ バッテリー表示は80%以上
✅ 時刻は午前9時や午後3時など標準的な時間
✅ 個人情報は表示しない（ダミーデータ使用）
```

---

### 方法2: Flutterのスクリーンショットツール使用

```bash
# デバイス接続を確認
flutter devices

# アプリを起動
flutter run --release

# 画面をキャプチャ（Flutter DevToolsから）
# または手動でスクリーンショット
```

---

### 方法3: Android Studioのエミュレーターで撮影

1. Android Studioを起動
2. AVD Manager でエミュレーター起動
3. アプリを実行
4. エミュレーター右側のツールバー → カメラアイコン
5. スクリーンショット保存

## 📏 画像要件

### サイズと解像度
```
最小サイズ: 320px
最大サイズ: 3840px
推奨サイズ: 1080 x 1920 px（縦向き）
          または 1920 x 1080 px（横向き）
フォーマット: PNG or JPG
```

### ファイル命名規則
```
screenshot_01_home.png         - ホーム画面
screenshot_02_settings.png     - 設定画面
screenshot_03_history.png      - 送信履歴画面
screenshot_04_signin.png       - サインイン画面（オプション）
```

---

## 🎨 画像編集（オプション）

スクリーンショットに枠や説明を追加する場合：

### 推奨ツール
- **Figma**（無料、オンライン）
- **Canva**（無料、オンライン）
- **Adobe Express**（無料）
- **GIMP**（無料、デスクトップ）

### 追加要素（任意）
- デバイスフレーム（iPhoneやAndroid端末の枠）
- タイトル/説明テキスト
  - 例: "簡単操作で水質報告"
  - 例: "送信履歴を一覧表示"
- ブランドカラー背景

## 📝 ダミーデータの使用

スクリーンショット撮影時は、個人情報を含まないダミーデータを使用：

### 推奨ダミーデータ
```
送信先メールアドレス: demo@example.com
測定時刻: 09:00
残留塩素値: 0.5 mg/L
送信者名: Demo User
```

### デバッグモードの活用
設定画面でデバッグモードを有効にすると：
- 1日1回制限なし
- テストデータで何度でも送信可能

---

## 🚀 撮影後の処理

1. **画像の確認**
   - サイズ要件を満たしているか
   - 画質は十分か（ぼやけていないか）
   - 個人情報が含まれていないか

2. **ファイル配置**
   ```
   playstore_assets/screenshots/
   ├── screenshot_01_home.png
   ├── screenshot_02_settings.png
   ├── screenshot_03_history.png
   └── screenshot_04_signin.png
   ```

3. **ファイルサイズ確認**
   - 各ファイル < 8MB（推奨 < 2MB）
