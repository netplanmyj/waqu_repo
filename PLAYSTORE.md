# Google Play Store 公開準備チェックリスト

水質報告アプリをGoogle Play Storeに公開するために必要な素材と準備項目の完全リストです。

## 📱 必須ビジュアル素材

### アプリアイコン
- **適応型アイコン (Adaptive Icon)**
  - フォアグラウンド: 432 x 432 px (透明背景PNG)
  - バックグラウンド: 432 x 432 px (PNG/ベクター)
  - 形状: 円形、正方形、角丸正方形に対応

- **レガシーアイコン**
  - 512 x 512 px (PNG、最大1MB)
  - 透明背景禁止

### スクリーンショット（必須）
- **スマートフォン用**: 最低2枚、最大8枚
  - 縦向き: 最小320px、最大3840px
  - 横向き: 最小320px、最大3840px
  - 推奨: 1080 x 1920 px または 1080 x 2340 px

- **7インチタブレット用**: 最低1枚、最大8枚
  - 縦向き: 最小320px、最大3840px
  - 横向き: 最小320px、最大3840px
  - 推奨: 1200 x 1920 px

- **10インチタブレット用**: 最低1枚、最大8枚
  - 推奨: 1800 x 2560 px

### プロモーション画像（オプション）
- **フィーチャー グラフィック**
  - 1024 x 500 px (JPG/PNG、最大1MB)
  - アプリのキービジュアル、テキストオーバーレイ可

- **プロモーション動画**
  - YouTube動画URL
  - 30秒〜2分推奨

## 📝 ストア掲載情報

### アプリの詳細
- **アプリ名**: 30文字以内
- **簡単な説明**: 80文字以内
- **詳細な説明**: 4000文字以内
- **カテゴリ**: ビジネス または ツール
- **タグ**: 最大5個

### プライバシーポリシー
- **プライバシーポリシーURL**: 必須
- **データ安全性**: ユーザーデータの取り扱いについて

### 年齢制限・コンテンツレーティング
- **対象年齢**: 全年齢対象推奨
- **コンテンツレーティング質問票**: 回答必須

水道関連業務の残留塩素濃度測定結果を、指定されたメールアドレスに自動送信するアプリです。
毎日の水質チェック業務を効率化し、確実な報告体制を構築できます。

✅ 主な機能
・残留塩素濃度の簡単入力
・測定結果の自動メール送信
・送信履歴の管理
・テスト送信機能
・個人専用Google Apps Script連携

🎯 対象ユーザー
・水道施設管理者
・水質検査担当者
・自治体職員
・水道工事業者

🔒 プライバシー
・個人情報は端末内にのみ保存
・外部サーバーへのデータ送信なし
