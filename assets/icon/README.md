# アイコン画像ディレクトリ

このディレクトリには、アプリアイコンのソース画像を配置します。

## 📁 必要なファイル

以下の3つのファイルを作成して配置してください：

### 1. icon.png (1024x1024px)
- **用途**: すべてのサイズのベースとなるマスター画像
- **形式**: PNG
- **背景**: 不透明（青色背景付き）
- **内容**: 水滴💧 + ブルー背景

### 2. icon_foreground.png (432x432px)
- **用途**: Android適応型アイコンの前景レイヤー
- **形式**: PNG
- **背景**: 透過（transparent）
- **内容**: 水滴💧のみ（白または明るいブルー）
- **重要**: 中央の288x288px円形領域内に主要素を配置

### 3. icon_background.png (オプション)
- **用途**: Android適応型アイコンの背景レイヤー
- **形式**: PNG
- **背景**: 不透明
- **内容**: 単色またはグラデーション
- **注**: 現在は `pubspec.yaml` で色指定（#2196F3）しているため不要

## 🎨 デザインツールでの作成方法

### オプション1: Canva（推奨・簡単）

1. https://www.canva.com/ にアクセス
2. 「カスタムサイズ」→ 1024x1024px
3. 背景色を `#2196F3` (青) に設定
4. 「素材」→「絵文字」→「💧」を検索
5. 水滴を配置、白色に変更
6. ダウンロード → PNG形式 → `icon.png` として保存

**フォアグラウンド用**:
1. 新規作成 → 432x432px
2. 背景を削除（透過）
3. 白い水滴のみ配置
4. ダウンロード → PNG形式 → `icon_foreground.png` として保存

### オプション2: Figma（プロフェッショナル）

1. https://www.figma.com/ にアクセス
2. 新規フレーム → 1024x1024px
3. 長方形ツールで背景作成（#2196F3）
4. ペンツールで水滴形状を描画
5. エクスポート → PNG → `icon.png`

### オプション3: オンラインアイコンジェネレーター

簡易的に作成したい場合：

1. https://icon.kitchen/ にアクセス
2. "Simple Shape" を選択
3. 水滴形状を選択または作成
4. 背景色を青系に設定
5. ダウンロード

## 🚀 作成後の手順

ファイルを配置したら、以下のコマンドを実行：

```bash
# 1. パッケージをインストール
flutter pub get

# 2. アイコンを自動生成
flutter pub run flutter_launcher_icons

# 3. 動作確認
flutter run
```
