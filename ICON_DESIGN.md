# アプリアイコンデザインガイド

## 🎨 デザインコンセプト

水質検査報告アプリのアイコンは、以下のコンセプトで設計します：

- **モチーフ**: 水滴💧
- **カラースキーム**: ブルー系（清潔感・水のイメージ）
- **スタイル**: シンプル、視認性重視

## 📐 必要なアセット

### 1. レガシーアイコン
- **サイズ**: 512x512px
- **形式**: PNG（透過なし、背景付き）
- **用途**: Google Play Store掲載用

### 2. 適応型アイコン（Adaptive Icon）
- **サイズ**: 432x432px（推奨：1024x1024pxで作成後、432x432pxに縮小）
- **形式**: PNG（透過あり）
- **構成**:
  - フォアグラウンド層: 水滴シンボル（透過背景）
  - バックグラウンド層: 単色またはグラデーション背景

### 3. iOS用アイコン
- **サイズ**: 1024x1024px
- **形式**: PNG（透過なし、角丸なし - OSが自動適用）
- **用途**: App Store Connect

## 🛠️ 作成手順

### ステップ1: デザインツールで作成

以下のいずれかのツールを使用：

1. **Figma** (推奨・無料)
   - https://www.figma.com/
   - ベクター編集可能
   - クラウド保存

2. **Canva** (簡単・無料)
   - https://www.canva.com/
   - テンプレート豊富
   - 初心者向け

3. **GIMP** (無料・高機能)
   - https://www.gimp.org/
   - Photoshop代替

4. **Adobe Illustrator/Photoshop** (プロ向け)
   - 有料だが高品質

### ステップ2: アイコンファイルの準備

作成したデザインを以下のファイル名で保存：

```
assets/
└── icon/
    ├── icon.png              # 1024x1024px マスターファイル
    ├── icon_legacy.png       # 512x512px レガシー用
    ├── icon_foreground.png   # 432x432px 適応型フォアグラウンド
    └── icon_background.png   # 432x432px 適応型バックグラウンド
```

### ステップ3: flutter_launcher_icons で自動生成

1. `pubspec.yaml` に設定追加
2. `flutter pub get`
3. `flutter pub run flutter_launcher_icons`

すべてのサイズバリエーションが自動生成されます。

## 💡 デザインのヒント

### 水滴の描き方

1. **シンプル版**: 涙型（上部に丸、下部に尖り）
2. **リアル版**: ハイライト（光の反射）を追加
3. **抽象版**: 3つの同心円で水の波紋を表現

### 配色のコツ

- メインカラーは青系統で統一
- コントラストを確保（水滴が背景に埋もれない）
- グラデーションは2色まで（複雑にしない）
- 光の反射は白または明るいブルーで表現

### サイズ対応

- 大きいサイズ(1024px)で作成し、縮小する
- 小さいサイズでも認識できるシンプルさを維持
- 細い線は避ける（小サイズで消える可能性）
- ディテールは最小限に

## 🎯 目指すゴール

- **認識性**: 一目で「水」に関するアプリとわかる
- **プロフェッショナル**: 業務用として信頼できる外観
- **シンプル**: 複雑すぎず、覚えやすいデザイン
- **ブランド**: 青を基調とした統一感のあるカラー
