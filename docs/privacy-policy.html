プライバシーポリシー

最終更新日: 2025年10月15日

水質検査報告アプリ（以下「本アプリ」）は、ユーザーの皆様のプライバシーを尊重し、個人情報の保護に努めています。本プライバシーポリシーは、本アプリがどのような情報を収集し、どのように使用・保護するかを説明するものです。

1. 収集する情報

1.1 ユーザーが提供する情報

本アプリは、以下の情報を収集します：

Googleアカウント情報

  • メールアドレス: Google Sign-Inで取得
  • アカウント名: Google Sign-Inで取得
  • プロフィール情報: Googleアカウントに登録されている基本情報

アプリ設定情報

  • 送信先メールアドレス: ユーザーが設定画面で入力
  • 地点番号: 測定地点の識別番号（例: 01）
  • デバッグモード設定: テスト送信の有効/無効

測定データ

  • 測定日: 水質検査を実施した日付
  • 測定時刻: 水質検査を実施した時刻
  • 残留塩素濃度: 測定された塩素濃度の値（mg/L）

2. 情報の使用目的

2.1 Gmail API経由のメール送信

  • 目的: 測定データを指定されたメールアドレスに送信
  • 使用スコープ: https://www.googleapis.com/auth/gmail.send
  • 送信元: ユーザー自身のGmailアカウント
  • 送信先: ユーザーが設定したメールアドレス
重要: 本アプリは、ユーザーのGmailアカウントを使用してメールを送信します。これは、ユーザー自身が送信者となることを意味します。第三者のサーバーやアカウントからメールが送信されることはありません。

3. 情報の保存場所と期間

3.1 ローカルストレージ（端末内）

以下の情報は、ユーザーの端末内のみに保存されます：

データ種類 保存場所 保存期間
アプリ設定 SharedPreferences ユーザーが削除するまで
送信履歴 SharedPreferences ユーザーが削除するまで
Gmailアクセストークン メモリ内 セッション終了まで

3.2 Firebase Functions（一時処理）

重要: Firebase Functionsはデータを永続的に保存しません。処理のためだけに一時的にデータを扱います。

データフロー：

  1. アプリからFunctionsにリクエスト送信
  2. Functionsが一時的にデータを受け取る
  3. Gmail API経由でメール送信
  4. 処理完了後、データは破棄

4. 情報の第三者提供

本アプリは、以下の場合を除き、ユーザーの個人情報を第三者に提供しません：

  • ユーザーの明示的な同意がある場合
  • 法令に基づく開示が必要な場合
  • 人の生命、身体または財産の保護のために必要な場合

4.1 使用する外部サービス

Google Gmail API

  • 提供元: Google LLC
  • 目的: メール送信機能の提供
  • スコープ: gmail.send（メール送信のみ）
  • プライバシーポリシー: Googleプライバシーポリシー

Firebase (Google Cloud)

  • 提供元: Google LLC
  • 目的: Firebase Functions（メール送信処理）、Firebase Auth（認証）
  • プライバシーとセキュリティ: Firebase プライバシーとセキュリティ

5. Gmail APIスコープの使用について

5.1 使用スコープ

本アプリは、以下のGmail APIスコープを使用します：

https://www.googleapis.com/auth/gmail.send

用途: 水質測定データをメールで送信するため

使用しないスコープ:

  • メールの読み取り（read）
  • メールの削除（delete）
  • メールの変更（modify）
  • ラベルの操作

5.2 権限の取り消し

ユーザーはいつでも権限を取り消すことができます：

  1. Googleアカウントのセキュリティページにアクセス
  2. 「水質検査報告アプリ」を選択
  3. 「アクセス権を削除」をクリック

6. データのセキュリティ

技術的保護措置

  • 通信の暗号化: すべてのAPI通信はHTTPS（TLS 1.2以上）で暗号化
  • 認証情報の保護: OAuth 2.0による安全な認証
  • ローカルデータの保護: Android標準の暗号化機能を使用

7. ユーザーの権利

データの訂正・削除権

ユーザーは、以下の方法でデータを管理できます：

  • 設定の変更: 設定画面から送信先メールアドレスを変更
  • 履歴の削除: 履歴画面で個別の履歴を長押し → 削除
  • アカウント連携の解除: アプリ内でサインアウト

8. 子どものプライバシー

本アプリは、13歳未満の子どもを対象としていません。13歳未満の子どもから意図的に個人情報を収集することはありません。
