# 💧 水質検査報告アプリ `waqu_repo`

**Water Quality Reporter** - 水道施設の残留塩素濃度測定結果をGmail経由で報告するAndroidアプリ

---

## アプリの目的

**水質検査報告アプリ `waqu_repo`** は、日本の水道施設で働く作業員が、**水道法に基づく残留塩素濃度の測定結果を施設管理者にメール報告する**ための業務用Androidアプリケーションです。

### 解決する課題

水道施設の作業員は、毎日の残留塩素濃度測定後、測定結果を施設管理者にメールで報告する必要があります。従来は以下の課題がありました：

- 手動でのメール作成が手間で時間がかかる
- 報告フォーマットが統一されず、ミスが発生しやすい
- 外出先からの報告が困難
- 送信履歴の管理ができない

本アプリは、これらの課題を解決し、**測定値を入力するだけで自動的にフォーマットされたメールを送信**できます。

### なぜGmail APIを使用するのか

本アプリは **Gmail API (gmail.send スコープ)** を使用して、作業員自身のGmailアカウントからメールを送信します。これには以下の理由があります：

1. **責任追跡と監査証跡**: 作業員の公式Gmailアカウントから送信することで、誰がいつ報告したかを明確に記録
2. **セキュリティ**: OAuth 2.0認証により、パスワードを保存せずに安全にメール送信
3. **送信確認**: Gmail の「送信済み」フォルダで送信履歴を確認可能
4. **法令遵守**: 水道法に基づく報告義務を確実に履行

本アプリは、**メールの送信のみ**を行います。メールの読み取り、変更、削除は一切行いません。

## アプリについて

水質検査報告アプリ `waqu_repo` は、水道関連業務の残留塩素濃度測定結果を、指定されたメールアドレスに自動送信するアプリです。毎日の水質チェック業務を効率化し、確実な報告体制を構築できます。

---

## 主な機能

### 📊 簡単入力
- 残留塩素濃度の測定値を数値入力
- 測定日時は自動記録
- 地点番号で測定場所を識別

### 📧 Gmail API経由でメール送信
- **Gmail API (gmail.send スコープ)** を使用
- ユーザー自身のGmailアカウントから送信
- OAuth 2.0による安全な認証
- 測定結果を自動フォーマットしてメール作成

### 📝 送信履歴管理
- 過去の送信履歴をアプリ内で確認
- 送信日時、宛先、送信結果を記録
- 端末内のみに保存（外部サーバーには保存しません）

### 🔒 セキュリティとプライバシー
- OAuth 2.0による安全な認証（パスワード不要）
- 測定データは端末内のみに保存
- Firebase Functions経由で安全にメール中継
- メールの読み取り・変更は一切行いません

---

## 使用する権限とその理由

### Gmail API (gmail.send)

本アプリは、以下の目的でGmail APIの **gmail.send スコープのみ** を使用します：

**使用目的**:
- 作業員のGmailアカウントから施設管理者へ測定報告メールを送信

**使用しないスコープ**:
- ❌ メール読み取り (gmail.readonly) - 使用しません
- ❌ メール変更 (gmail.modify) - 使用しません
- ❌ 連絡先アクセス - 使用しません
- ❌ その他のGoogleサービス - 使用しません

**データフロー**:
```
1. 作業員が測定値を入力
2. アプリが測定結果をフォーマット
3. Gmail API (gmail.send) で作業員のGmailから送信
4. 施設管理者がメールを受信
5. 送信履歴を端末内に保存
```

**重要**: メール内容はサーバーに保存されません。Firebase Functionsは中継のみを行い、ログも残しません。

## プライバシーとセキュリティ

本アプリは、ユーザーのプライバシーを最優先に考えています：

### データ保護
- ✅ 測定データは端末内のみに保存（SharedPreferences使用）
- ✅ メール内容はサーバーに保存されません
- ✅ Firebase Functions は中継のみ（ログなし）
- ✅ 第三者へのデータ提供は一切ありません

### 安全な認証
- ✅ OAuth 2.0による安全な認証
- ✅ パスワードの保存は不要
- ✅ ユーザーはいつでも権限を取り消し可能

### 透明性
- ✅ 送信前に宛先を確認可能
- ✅ 送信履歴をアプリ内で確認可能
- ✅ Gmail の「送信済み」フォルダで確認可能

詳細は [プライバシーポリシー](/waqu/privacy-policy) をご覧ください。
